nth-child()

nth-child() and all other similar pseudo classes are using an+b as the formula to select the child elements. But CSS also gives us handy even and odd which are used to select all the odd and even children respectively.

See the below example, all the even elements are in this color and odd elements are in this color.

Now lets see how we can leverage this pseduo class using an+b to select the children in a way we never did before.

In the below example, I've applied this color for every 3rd element starting from 2nd child element. To achieve this I used 3n+2. Replace n with 0, 1, 2, 3, 4... until 12 and you'll get to know the result will be 2, 5, 8 and 11.
